Traits QuickStart

Traits description

Architecture

Traits is structure in a similar fashion to the Helix Architecture with a few changes.

  • The projects node is called Apps as it will be storing a variety of application types

Apps

The Apps section stores folders with various applications. Each application will reference elements of the foundation and various features. Each application will contain its own documentation folder.

Features

The Features section contains libraries that extend the functionality of HubAI. A library is placed in the feature section if that library is not likely to be referenced by another HubAI Library.

Foundation

The Foundation section contains the building blocks of HubAI. These libraries can reference each other but none of the libraries in the feature section
